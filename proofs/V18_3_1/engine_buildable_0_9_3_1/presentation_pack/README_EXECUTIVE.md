# OBSIDIA — Executive Pack (10 minutes)

## What this is
Obsidia is a deterministic governance infrastructure that decides **before execution**.
It returns one of three outcomes: **BLOCK | HOLD | ACT**.
It is built to be auditable and provable (append-only logs, hash chain, WORM storage, daily attestation).
